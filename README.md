## Description
Air Canvas is a computer vision project that allows users to draw on the screen
using hand gestures captured via a webcam.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How It Works
- Webcam captures hand movements
- Index finger is tracked
- Drawing is done in real time on a virtual canvas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the program:
   python air_canvas.py
